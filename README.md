# project-hangman
# Hangman Game – Car Brands Edition

A simple **console-based Hangman game** built in **Python** where players guess the names of famous car brands one letter at a time. This project was created to practice fundamental Python programming concepts such as loops, functions, conditionals, lists, and user input.

---

## About the Project

In this game, a random car brand is selected from a predefined list. The player must guess the word by entering one letter at a time. Each incorrect guess reduces the number of remaining attempts. The game ends when the player successfully guesses the word or runs out of guesses.

This project is beginner-friendly and demonstrates the implementation of a classic Hangman game using only Python's built-in libraries.

---

## Features

- Randomly selects a car brand each game
- Guess one letter at a time
- Validates user input
- Prevents duplicate guesses
- Six incorrect guesses allowed
- Win/Lose game messages
- Option to play again
